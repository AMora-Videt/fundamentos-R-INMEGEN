# Implementación de Análisis Estadístico y Visualización de Datos en R

## Contexto Académico
Este repositorio constituye el registro técnico y metodológico de la formación inicial en ciencia de datos aplicada, desarrollada en el marco del curso "Bases de Programación en R" impartido por el Instituto Nacional de Medicina Genómica (INMEGEN). 

El objetivo primordial de este espacio es documentar la transición hacia flujos de trabajo computacionales orientados a la medicina basada en evidencia y la investigación clínica.

## Competencias Técnicas Demostradas
A través del análisis exploratorio contenido en este proyecto, se implementan las siguientes capacidades técnicas:

* **Gestión de Datos Masivos**: Utilización de la librería `vroom` para la ingesta eficiente de conjuntos de datos y `tidyverse` (`dplyr`) para la manipulación y depuración de estructuras.
* **Análisis Estadístico Visual**: Aplicación de escalas logarítmicas (`scale_y_log10`) para la normalización visual de distribuciones con alta variabilidad y sesgo, facilitando la interpretación de datos biológicos o epidemiológicos.
* **Evaluación de Correlaciones**: Construcción de matrices de correlación y visualización mediante mapas de calor para identificar asociaciones significativas entre variables.
* **Detección de Anomalías**: Implementación de gráficos de caja (boxplots) para la identificación técnica de valores atípicos (outliers) y la evaluación de la dispersión de la muestra.

## Proyección en Investigación
Este proyecto sirve como fundamento metodológico para la ejecución de futuras revisiones sistemáticas y metanálisis. La estructura aquí presentada prioriza la reproducibilidad y el rigor en el reporte de datos, alineándose con los estándares internacionales de transparencia científica.
